# ステータス・タブ早見表

| 機能 | 種類 | 値（順序どおり） | 補足 |
|:--|:--|:--|:--|
| 商品管理 | 一覧タブ | すべて / 公開中 / 下書き / アーカイブ済み | 「アーカイブ済み」はナビ「商品管理 > 全タブ」から確認できる |
| 商品管理 | ステータス（作成時） | 公開 / 下書き / 非公開 | 「非公開」で作成した商品は編集画面では「アーカイブ済み」と表示される |
| 商品管理 | ステータス（編集時） | 公開中 / 下書き | 作成時の「非公開」はこの画面には表示されない |
| 移動伝票 | 一覧タブ | すべて / 出荷作業 / 一部受領済み / 受領済み / キャンセル | ステータスを直接進めるボタンはなく、出荷・入荷実績登録に連動して自動変化する。出荷前入荷、部分入荷、過入荷、在庫0からの移動/出荷も実機確認済み（2026-06-18） |
| 調整伝票 | 一覧タブ | すべて / 未実施 / 実施済み / キャンセル | 「実行」ボタンで「未実施→実施済み」になる。未実施の「キャンセル」は確認ダイアログなしで即時キャンセル。実施済みでも「キャンセル」ボタンは表示されるが、押すとキャンセル不可トーストが出て状態は変わらない。キャンセル済みでも実行ボタンが残る場合があるが実行はブロックされる |
| 調整伝票 | 理由（選択肢） | 廃棄 / 見本 / 紛失 / 棚卸差異 / その他 | 5種類。「破損」「検品」「予備」は選択肢にない。2026-06-18再確認では未選択のまま保存すると「理由を選択してください」で保存不可 |
| 取置伝票 | 一覧タブ | すべて / 未処理 / 処理済み | 未処理詳細の「処理済みとしてマークする」→確認ダイアログ「実行する」で処理済みになり、取置中在庫が解放される。数量0は保存不可 |
| 在庫依頼 | 一覧タブ | 確認待ち / 確保済み / 終了 | 「在庫依頼をクローズする」で終了になる。確保済み後はクローズ不可。確保済み一覧から作成した移動伝票は外部システム連携扱いになり、完了後も元依頼は確保済みのまま |
| 出荷管理 | 一覧タブ | 保留中 / 出荷待ち / 依頼済み / 作業中 / 欠品・要対応 / 出荷完了 | キャンセルタブなし。移動伝票キャンセル後の出荷指示は一覧から消えるが、関連リンクや直接URLで詳細を開ける場合がある |
| 出荷管理 | エクスポートの出荷作業ステータス（選択肢） | 指定しない / 出荷待ち / 保留中 / 依頼済み / 作業中 / 欠品・要対応 / キャンセル済み / 出荷完了 | 「条件指定でエクスポート」フォームの絞り込み選択肢（8種） |
| 入荷管理 | 一覧タブ | 入荷待ち / 入荷依頼済み / 入荷作業中 / 要対応 / 入荷完了 / キャンセル | 移動伝票キャンセル後の入荷指示は「キャンセル」タブに「入荷キャンセル」バッジで残る。出荷前入荷、部分入荷、過入荷、手動完了を実機確認済み（2026-06-18） |
| 注文管理 | 一覧タブ | すべて | タブは「すべて」のみ（ステータス別タブなし）。絞り込みはテナント・タグ・チャネルで行う |

| 返品（注文管理 > 返品） | 一覧タブ | タブなし | 絞り込みはキャンセル・交換出荷で行う |

| 顧客管理 | 一覧タブ | すべて | タブは「すべて」のみ。絞り込みはテナント・メタフィールド・会員証バーコードの3種 |

| 在庫区分（詳細画面・9列） | 列構成 | 販売可能 / 引当済み / 取置中 / 破損 / 検品 / 予備 / 手持ち / 積送中 / 入荷予定 | 在庫詳細のロケーション別テーブル列（2026-06-14更新）。手持ち = 販売可能+引当済み+取置中+破損+検品+予備（積送中・入荷予定は手持ち外）。移動伝票の作成/出荷/入荷で積送中・入荷予定が動くことを実機確認済み（2026-06-18） |
