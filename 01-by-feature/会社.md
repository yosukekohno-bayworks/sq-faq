# 会社

> 対象画面: 会社 / /admin/companies　|　最終確認: 2026-06-15

## この機能でできること

- 法人顧客（会社）の新規作成・一覧管理
- 会社ごとのロケーション（納品先住所）の管理
- 会社ごとの担当者（発注確認メールの送付先）の管理
- 会社ごとの注文履歴・売上の確認

---

## 会社一覧（/admin/companies）

### タブ

| タブ名 | 表示対象 |
|:--|:--|
| すべて | 全会社 |

### テーブル列

| 列名 | 内容 |
|:--|:--|
| （チェックボックス） | アイテムを選択する（一括選択用） |
| 会社 | 会社名 |
| ロケーション | 登録されているロケーション数（例：「1箇所のロケーション」） |
| 総注文数 | 注文件数（例：「0個の注文」） |
| 販売合計 | 累計金額（例：「¥0」） |
| 作成日 | 登録日時（例：「2026年06月06日 17:39」） |

行全体をクリックすると会社詳細画面へ遷移します。

### 主な操作

| 要素（UIラベル） | 種別 | 操作すると何が起こるか |
|:--|:--|:--|
| インポート | ボタン | クリックしても現状何も起きない（機能未実装） |
| 会社を作成する | リンクボタン | `/admin/companies/create`（会社作成フォーム）へ遷移 |
| 前へ／次へ | ページネーションボタン | 前後ページへ移動 |

> **注意:** 「インポート」ボタンはクリックしてもドロップダウン・モーダル・画面遷移のいずれも発生しません。現時点ではこの操作はできません。

> **注意:** 会社一覧で行選択しても、会社本体を削除する通常操作は確認できていません。削除導線は表示・実行できない状態です。

## 会社作成フォーム（/admin/companies/create）

![「会社を作成する」フォーム（会社情報・担当者・ロケーション・配送先住所・請求先住所）](../_analysis/screenshots/companies-create-form.png)

「会社を作成する」ボタンから遷移します。入力後「保存する」で会社が作成されます。

### 会社情報

| 項目（UIラベル） | 必須 | 補足 |
|:--|:--|:--|
| 会社名 | 必須（*） | 請求書・配送ラベルに表示されます |
| 会社ID | 任意 | 外部IDがある場合に登録します |
| 担当者 | 任意 | 検索テキストボックスで顧客を紐付けます |

### ロケーション（配送先）

会社を作成する際に、最初のロケーション（納品先）を同時に登録します。

| 項目（UIラベル） | 必須 | 補足 |
|:--|:--|:--|
| ロケーション名 | 必須（*） | 例：「東京本社」 |
| ロケーションID | 任意 | 外部IDがある場合に登録します |
| コード | 任意 | プレースホルダー: 12345 |

### 配送先住所

| 項目（UIラベル） | 必須 | バリデーションエラー文言 |
|:--|:--|:--|
| 国/地域 | 必須（*） | 国を選択してください |
| 性 | 必須（*） | 苗字を入力してください |
| 名 | 必須（*） | 名前を入力してください |
| 会社 | 任意 | — |
| 郵便番号 | 必須（*） | 郵便番号を入力してください |
| 都道府県 | 必須（*） | 都道府県を入力してください |
| 市区町村 | 必須（*） | 市区町村を入力してください |
| 住所 | 必須（*） | 住所を入力してください |
| 建物名、部屋番号など | 任意 | — |
| 電話番号 | 任意 | — |

### 請求先住所

「配送先住所と同じ」チェックボックスがあり、デフォルトでチェック済みです。チェックを外すと配送先住所と同一の項目が展開され、個別に入力できます。

## 会社詳細画面（/admin/companies/{id}）

![会社詳細画面（直近の注文・ロケーション・担当者と右サイドバー）](../_analysis/screenshots/gap-company-detail-top.png)

### ページヘッダー

- 会社名が大見出しで表示されます
- 「2026年XX月XX日から顧客」（作成日）がサブテキストとして表示されます

### 左カラム（メインコンテンツ）

| セクション | 内容 |
|:--|:--|
| 直近の注文 | 注文一覧。注文がない場合は「この会社はまだ注文していません」と表示されます |
| ロケーション | 登録済みロケーションのテーブル（ロケーション名 / 売上 / 注文）。「ロケーションを追加」ボタンあり |
| 担当者 | 登録済み担当者のテーブル（名前 / メールアドレス）。「担当者を追加」ボタンあり |

### 右サイドバー

| セクション | 内容 |
|:--|:--|
| 会社名 | 会社名テキスト。編集ボタン（ペンアイコン）で変更できます |
| 会社ID | 外部ID。未設定の場合は「未設定」と表示されます |
| メモ | 会社に関するメモ。編集ボタンで変更できます |

## ロケーション追加（/admin/companies/{id}/locations/create）

会社詳細の「ロケーションを追加」ボタンから遷移します。入力項目は会社作成フォームのロケーション・配送先住所セクションと同一です。

### ロケーション情報

| 項目（UIラベル） | 必須 | 補足 |
|:--|:--|:--|
| ロケーション名 | 必須（*） | 「注文のロケーションを特定するために使用します。例：ミッドタウン」 |
| ロケーションID | 任意 | 「会社のロケーションを識別する外部IDがある場合に登録することができます。」 |
| コード | 任意 | プレースホルダー: 12345 |

続いて「配送先住所」と「請求先住所」（デフォルトで配送先と同じ）を入力します。

## 担当者の追加

![「担当者を作成する」ダイアログ（姓・名・メールアドレス・電話番号）](../_analysis/screenshots/gap-company-detail-add-contact-dialog.png)

会社詳細の「担当者を追加」ボタンを押すと「担当者を作成する」ダイアログが開きます。

### 担当者を作成するダイアログの入力項目

| 項目（UIラベル） | 必須 | プレースホルダー |
|:--|:--|:--|
| 姓 | 必須（*） | 山田 |
| 名 | 必須（*） | 花子 |
| メールアドレス | 必須（*） | you@company.com |
| 電話番号 | 任意 | 09012345678 |

> 「発注の確認メールなどはこのメールアドレスに配信されます。」

ダイアログのボタンは「キャンセル」と「保存する」です。

**注意:** 担当者は顧客管理（/admin/purchasing_customers）の顧客とは別のエンティティです。担当者を作成しても顧客管理の一覧には表示されません。

### 担当者詳細画面

担当者の行をクリックすると担当者詳細画面へ遷移します。現時点では担当者名・作成日時のみが表示されており、情報の編集や詳細確認はこの画面からはできません。<!-- TODO: 要確認（担当者詳細画面の今後の機能追加） -->

### 担当者の削除

担当者一覧で行を選択すると「担当者を削除」が表示されます。確認ダイアログは「担当者を削除する」で、本文は「選択されている1件の担当者を削除しますか？」です。削除後、対象の担当者行は一覧から消えます。

## ロケーション詳細画面（/admin/companies/{id}/locations/{location_id}）

### 左カラム

| セクション | 内容 |
|:--|:--|
| 直近の注文 | 注文がない場合は「このロケーションからはまだ注文がありません」と表示されます |

### 右サイドバー

| セクション | 内容 |
|:--|:--|
| ロケーション名 | 編集ボタン（ペンアイコン）で変更できます |
| ロケーションID | 外部ID。未設定の場合は「未設定」 |
| コード | 未設定の場合は「未設定」 |
| 配送先住所 | 登録済みの住所を表示します |
| 請求先住所 | 配送先と同一の場合は同じ内容を表示します |
| メモ | 編集ボタンで変更できます |

### ロケーションの削除

会社ロケーション詳細の三点メニューで確認できる操作は「ロケーションを編集」のみです。会社ロケーションを削除するUIは確認できていません。

## 補足・注意点

- 顧客管理（/admin/purchasing_customers）の顧客は販売チャネル経由で自動登録されますが、「会社」は管理画面から直接作成できます。
- 会社の「担当者」は発注確認メールの宛先であり、顧客管理の購入顧客とは別データです。
- 「インポート」ボタンは現時点では機能していません（未実装）。
- 「卸売」（/admin/b2b）は会社とは別機能で、現時点では未実装（「TODO」表示のみ）です。
