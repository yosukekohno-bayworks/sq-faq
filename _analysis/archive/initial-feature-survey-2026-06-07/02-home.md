# ホーム画面

> 調査日: 2026-06-05 | URL: `/admin` | 読み取り専用

## 画面目的
管理画面のダッシュボード。主要機能へのクイックアクセスと外部リンクを提供する。

## 主要構成要素

### ヘッダー
- 左上: SQ ロゴ（`/admin`リンク）
- 右上: 組織名「stack-ps-yosuke」 + ユーザー名「陽介 河野」ボタン（アバター付き）
- ハンバーガーメニューボタン（サイドナビ開閉と推測）

### クイックリンクカード（7枚）
| カード見出し | リンク先 | 説明文（原文） |
|:--|:--|:--|
| 商品管理 | `/admin/products` | 商品の追加・編集、付加情報の管理、カタログの作成ができます。 |
| 在庫管理 | `/admin/inventory_items` | 在庫の移動、仕入れ、在庫の状況を確認することができます。 |
| 注文管理 | `/admin/orders` | 販売チャネルで作成された注文の管理、注文の追跡、顧客への通知ができます。 |
| 顧客管理 | `/admin/purchasing_customers` | 顧客情報の管理、顧客の注文履歴、顧客の分析ができます。 |
| CSVでデータをインポートする | `/admin/csv_import` | CSVファイルを使ってデータの書き込み・編集ができます。 |
| CSVでデータをエクスポートする | `/admin/csv_export` | 指定のデータをCSV形式でエクスポートすることができます |
| PDFでデータをエクスポートする | `/admin/pdf_export` | 指定のデータをPDF形式でエクスポートすることができます |

### 外部リンクセクション
| ラベル | URL | 説明文（原文） |
|:--|:--|:--|
| 製品アップデートを見る | `https://stack-inc.notion.site/...` | 最新リリース、改善点、修正点をご覧いただけます。 |
| ヘルプセンターを見る | `https://docs.sqstack.com/docs/guide/introduction` | 各種機能の使い方ガイドをご覧いただけます。 |

### フッター
- 「Proudly built by Stack Inc」

## サイドナビ構成（全24項目確認済み）
- **メイン**: ホーム / 商品管理 / 在庫管理 / 注文管理 / 顧客管理 / 発注管理 / 販売設定 / 会計 / 分析
- **オペレーション**: 入荷管理 / 出荷管理 / 在庫依頼
- **CRM**: ディスカウント / ポイント / 会員ランク
- **販売チャネル**: Shopify / OmnibusCore / スマレジ / リテールポータル / 卸売
- **その他**: 設定
