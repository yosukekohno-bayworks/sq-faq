## 追加で確定した挙動

- 会社を作成しても、`/admin/purchasing_customers` は空のまま。会社の「担当者/ロケーション」と購買顧客は別。
- 会社と商品を作成しても、`/admin/draft_orders` の「注文を作成する」は `<a>` タグの disabled CSS 状態のまま。`href` なし、`pointer-events:none`。
- 会社と商品を作成しても、`/admin/sale_change_line_items` の「売上実績を作成する」は disabled CSS 状態のまま。
- `/admin/purchasing_customers/create` と `/admin/draft_orders/create` は引き続きアプリ内「予期せぬエラー」。
- `/admin/order_returns/create` と `/admin/sale_change_line_items/create` は404。
- 発注伝票は、取引先/テナント/通貨/SKU/単価/数量まで入力できるが、作成時に `エラーが発生しました。しばらくしてから再度お試しください` で保存不可。発注経由の入荷連携は未実証のまま。
