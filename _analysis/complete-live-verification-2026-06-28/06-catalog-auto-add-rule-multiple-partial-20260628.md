# 06 カタログ: 自動追加ルールの複数ルール・部分一致 2026-06-28

## 対象

- 検証用カタログ: `TEST_FAQ_RULE_MATRIX_20260628_011423`
- カタログURL: `/admin/catalogs/2fc5f58c-49ca-5a95-9f69-3c4ab24f5510_Catalog`
- ルール1: `製造元` / `一致する` / `TEST_VENDOR_A_20260628_011423`
- ルール2: `製造元` / `一致する` / `TEST_VENDOR_B_20260628_011423`
- ルール3: `製造元` / `一致する` / `TEST_VENDOR_PREFIX_20260628_011423`

## 検証商品

- A一致商品
  - 商品名: `TEST_FAQ_RULE_OR_A_20260628_011423`
  - 商品コード: `TEST_FAQ_RULE_OR_A_011423`
  - 製造元: `TEST_VENDOR_A_20260628_011423`
  - URL: `/admin/products/6980895c-1bf3-5093-b304-c8b939165bca_Product`
- prefix部分一致商品
  - 商品名: `TEST_FAQ_RULE_PARTIAL_20260628_011423`
  - 商品コード: `TEST_FAQ_RULE_PARTIAL_011423`
  - 製造元: `TEST_VENDOR_PREFIX_20260628_011423_EXTRA`
  - URL: `/admin/products/69f83413-e854-54f0-9346-0d28a69cd2ca_Product`

## 結果

- 同一項目 `製造元` のルールを3件登録できた。
- `製造元=TEST_VENDOR_A_20260628_011423` の新規商品はカタログへ自動追加された。
- `製造元=TEST_VENDOR_PREFIX_20260628_011423_EXTRA` の新規商品は、ルール値 `TEST_VENDOR_PREFIX_20260628_011423` とは前方一致するが、カタログへ自動追加されなかった。
- カタログの商品タブには `TEST_FAQ_RULE_OR_A_20260628_011423` のみ表示された。
- 検証用商品2件と検証用カタログは削除済み。
- 削除済みURL再確認:
  - A一致商品: `該当するProductが見つかりませんでした。`
  - prefix部分一致商品: `該当するProductが見つかりませんでした。`
  - 検証用カタログ: `該当するCatalogが見つかりませんでした。`

## 判定

- 同一項目の複数ルールは、確認範囲ではOR条件として動く。少なくとも3ルール中1件に一致する新規商品は自動追加される。
- `一致する` は前方一致ではなく、確認範囲では完全一致として扱われる。
- 未確認として残すもの: ブランドコードとの異種項目混在時の結合、カンマ等の複数値入力、`*` などを入れたワイルドカード文字列の扱い。
