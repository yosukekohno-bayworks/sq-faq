# 02 管理メンバー権限保存 実機確認 2026-06-28

## 対象

- URL: `/admin/settings/users`
- 対象: 既存管理メンバー `河野 陽介`
- 操作: 詳細画面の「権限」セクションで、権限グループを変更せずに `保存する` を押下

## 結果

- 管理メンバー詳細には `基本情報`、`ユーザーテナント`、`権限` の各セクションがある。
- `権限` セクションの保存ボタンラベルは `保存する`。
- 変更なしで押下しても保存処理は実行され、トースト `ユーザー権限を更新しました` が表示された。
- `適用されている権限一覧: 76件` の表示を確認。

## 注意

権限グループを別のグループへ変更した場合の画面差分・別権限ユーザーでのログイン時の実効制御は未確認。
