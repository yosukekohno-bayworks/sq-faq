# ロケーション 場所種別選択肢 実機確認 2026-06-28

- 対象URL: `/admin/settings/locations/create`
- h1: `ロケーションを作成`
- 操作: 保存せず、作成フォームのselect要素を読み取り

## 結果

| 項目 | 結果 |
|:--|:--|
| 場所種別の選択肢 | `倉庫` / `店舗` |
| 場所種別の内部値 | `WAREHOUSE` / `RETAIL` |

## 判断

- 2026-06-28時点のロケーション作成フォームで、場所種別は `倉庫` / `店舗` の2種のみ。
- `選択してください` はdisabledのプレースホルダーで、実選択肢には含めない。
- この確認はロケーション作成フォームの選択肢確認であり、各業務画面での使われ方は各画面の個別検証に従う。
