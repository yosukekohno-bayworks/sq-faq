# 06 カタログ: 同名作成可否と削除 2026-06-28

## 対象

- 画面: カタログ一覧 / カタログ作成 / カタログ詳細
- 既存名: `TEST_FAQ_カタログ001`
- 新規作成で入力した同名: `TEST_FAQ_カタログ001`

## 操作

1. `/admin/catalogs` で既存カタログ `TEST_FAQ_カタログ001` を確認。
2. `カタログを作成する` を開く。
3. タイトルに同名 `TEST_FAQ_カタログ001` を入力して `保存する`。
4. 作成された新規カタログ詳細の `その他の操作` から `カタログを削除する` を実行。

## 結果

- 同名タイトルでも保存できた。
- 保存後の遷移先: `/admin/catalogs/bb1dfb95-61b5-533f-9f27-f6083868474d_Catalog`
- 作成後トーストは `商品を作成しました` と表示された（文言はカタログ画面としては不自然だが、カタログ詳細へ遷移）。
- 詳細画面には `その他の操作` > `カタログを削除する` がある。
- 削除確認ダイアログは `カタログを削除する`、本文は `カタログを削除します。この処理は巻き戻すことができません。`。
- 削除後はカタログ一覧へ戻り、トースト `カタログを削除しました` が表示された。
- 検証用に作成した重複カタログは削除済み。

## 判定

カタログ名は一意制約ではなく、同名カタログを複数作成できる。運用上は用途が分かる名前を付ける必要がある。カタログ詳細から削除可能。
