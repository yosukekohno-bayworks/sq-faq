# 06 カタログ: 自動追加ルールの実効・削除後挙動 2026-06-28

## 対象

- 検証用カタログ: `TEST_FAQ_AUTORULE_20260628_010235`
- カタログURL: `/admin/catalogs/f646c313-8ae5-5771-8f52-c0fda4b71f74_Catalog`
- 自動追加ルール: `項目=製造元` / `条件=一致する` / `値=GU`
- 検証用商品: `TEST_FAQ_AUTORULE_PRODUCT_20260628_010235`
- 商品コード: `TEST_FAQ_AUTORULE_PRODUCT_010235`
- 商品URL: `/admin/products/3322fe51-7ab8-50b4-902f-d888a29d69ce_Product`

## 操作と結果

1. 検証用カタログを作成。
   - 作成直後の商品タブは空。
2. 自動追加ルール画面で `項目=製造元` / `条件=一致する` / `値=GU` を追加。
   - ルール一覧に `製造元 / 一致する / GU` と表示。
   - 既存の製造元 `GU` 商品は、ルール作成直後の商品タブには追加されなかった。
3. ルール作成後に、製造元 `GU` の検証用商品を新規作成。
   - カタログの商品タブに検証用商品が自動追加された。
4. 自動追加ルールを削除。
   - 行選択後に `ルールを削除する` が表示。
   - 確認ダイアログ本文は `選択された1件のルールを削除します。この処理は巻き戻すことができません。`。
   - 削除後トーストは `ルールを削除しました`。
5. ルール削除後にカタログ商品タブを再確認。
   - 自動追加済みの検証用商品はカタログに残った。
6. 検証用商品を削除。
   - 削除後、検証用カタログの商品タブは空になった。
7. 検証用カタログを削除。
   - 削除後トーストは `カタログを削除しました`。
8. 削除済み直URLを再確認。
   - 商品URLは `該当するProductが見つかりませんでした。`。
   - カタログURLは `該当するCatalogが見つかりませんでした。`。

## 判定

- 自動追加ルールは、少なくともルール作成後に作成された一致商品をカタログへ追加する。
- ルール作成時点で既に存在する一致商品には、確認範囲では即時遡及しない。
- 自動追加ルールを削除しても、すでに追加済みの商品はカタログに残る。
- 商品本体を削除すると、カタログ内の商品行も消える。
- 検証用商品・検証用カタログは削除済み。
