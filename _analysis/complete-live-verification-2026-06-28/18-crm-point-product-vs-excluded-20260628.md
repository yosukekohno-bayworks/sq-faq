# CRM: 商品倍率0倍と利用外商品の切り分け 2026-06-28

対象:

- 注文ポイント付与ルール `TEST_FAQ_注文ポイント付与ルール`
- 商品倍率タブ `/admin/point_calculation_rules/{id}/point_multiplier_products`
- 利用外商品 `/admin/point_application_excluded_products`

## 商品倍率

- 「追加する」から `商品倍率を追加する` ダイアログが開く。
- 項目は `商品` と `倍率`。
- 商品選択ダイアログの列は `商品 / 商品コード`。
- `TEST_20260622_OPTION` を選び、倍率 `0` で保存できた。
- 保存後の一覧は `商品 / 倍率` 列で、対象行は `0倍` 表示。
- 行選択後に削除でき、確認ダイアログは `商品倍率を削除しますか？`。
- 検証用追加行は削除済み。

## 利用外商品

- `/admin/point_application_excluded_products/create` は `商品を選択する` のみ。
- 商品選択ダイアログの列は `商品 / 商品コード`。
- `TEST_20260622_OPTION` を保存できた。
- 保存後の一覧は `商品 / 作成日時` 列。
- 行選択後に削除でき、確認ダイアログは `利用対象外商品を削除しますか？`。
- 検証用追加行は削除済み。

## 結論

商品倍率0倍は注文ポイント付与ルール内の付与倍率設定。利用外商品は別画面で管理するポイント利用側の設定。どちらも商品単位のUIで、SKUコード列は表示されない。
